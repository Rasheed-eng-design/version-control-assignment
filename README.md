# version-control-assignment

## What is Version Control?
Version control is a system that tracks changes in files over time. It allows developers to go back to previous versions, collaborate with others, and avoid losing work.

---

## Difference Between Git and GitHub

- **Git**: A tool installed on your computer used to track changes in code.
- **GitHub**: A cloud platform where Git repositories are stored and shared.

## 3 GitHub Alternatives

## Difference Between git fetch and git pull

- **git fetch**: Downloads changes from remote but does NOT merge them.
- **git pull**: Downloads changes AND merges them into your current branch.

---

## What is git rebase?

Rebase moves your branch to a new base commit, making history cleaner.
